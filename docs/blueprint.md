# **App Name**: AuthZen

## Core Features:

- Registration: Allow new users to create accounts using their email address and a password.
- Login: Allow existing users to log in using their registered email address and password.
- User Authentication: Verify user credentials against Firebase authentication service.
- Session Management: Maintain user sessions to keep users logged in during navigation.
- Personalized Greeting: Display a welcome message that includes the logged-in user's email address.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calm and inviting feel.
- Background color: Pale grayish-lavender (#F5F5FF).
- Accent color: Muted plum (#9467BD) for interactive elements.
- Font: 'PT Sans', a humanist sans-serif, for both body and headline text. 
- Clean, centered layout for easy readability and focus on authentication forms and user greeting.
